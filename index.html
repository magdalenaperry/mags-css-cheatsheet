CSS cheatsheet
css cheatsheet used to create websites with navigation bars and flex-boxes.

@media queries

Add a media query for screens that are less than 768px and need to take up the entire screen width. 

                
@media screen and (max-width:
768px){
    .section{
        width: 100%;
    }
}

flexbox

Create a main container that holds rows of cards within a flexbox.

                
.main-container {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: normal;
    align-items: normal;
    align-content: normal;
}

pseudo-elements

Add a tooltip to hover directly over an element.

                
.tooltip{
    text-decoration: underline;
    position:relative;
}
.tooltip:hover::before{
    content: attr(data=tool-tip);
    display: inline-block;
    position: absolute;
    top: 0;
    left: 50%;
    transform: translate(-50%, 
    -100%);
    width:100%;
}

CSS selectors

Select every a element whose href attribute begins with "https". Selects the first element that is placed directly after the first element. 

                
a[href^="https"]{
    background: white;
}
div + p {
    color: white;
}

gradients

Add a gradient as a background image.

                
.classname {
background-image: 
linear-gradient(color, color, color, 
color, etc);
}

vertical nav bars

Create a vertical navigation bar.

                
ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
    width: 200px;
    background-color: #f1f1f1;
}
li a {
    display: block;
    padding: 8px 16px;
    text-decoration: none;
    }
li a:hover {
    background-color: #555;
    color: white;
    }
